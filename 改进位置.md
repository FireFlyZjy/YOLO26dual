## 2026-05-14

### 新增 AIFI Transformer自注意力模块

> 配置目录: `ultralytics/cfg/models/26-RGBT/2026-05-14/`
> 来源: RT-DETR, yolo-improve/yolov5-AIFI.py

基于 `yolo26-RGBT-midfusion.yaml`，将 SPPF 替换为带2D sin-cos位置编码的Transformer自注意力。

**文件修改对照表**:

| 配置文件 | 模块 | 改动 |
|----------|------|------|
| `yolo26-RGBT-midfusion-AIFI.yaml` | AIFI Transformer | 修改1行 |

**Backbone改动**:

```
# 原 (行49):
- [-1, 1, SPPF, [1024, 5, 3, True]]  # 24-SPPF

# 新 (行49):
- [-1, 1, Att_AIFI, [1024, 8, 0.0]]  # 24-AIFI: 带位置编码的全局自注意力
```

**Head无改动**: AIFI与SPPF输入输出维度一致，所有Head引用保持不变。

### 代码文件修改

| 文件 | 改动 |
|------|------|
| `ultralytics/nn/ExtraModules/common.py` | 末尾追加 Att_AIFI 和 Att_ScalSeq 两个类，带 `(2026-05-14)` 日期标记 |
| `ultralytics/nn/tasks.py` | base_modules 追加 Att_AIFI, Att_ScalSeq 注册，带日期和功能注释 |
| `train_RGBT_midfuse_testAddModules_withResults.py` | 新建，训练+验证一体化脚本 |

### AIFI 架构示意

```
        P5 Concat (可见光 + 红外)
              |
    ┌─────────────────────────┐
    │   Att_AIFI (替换SPPF)    │  ← MultiheadAttention + FFN
    │   2D sin-cos 位置编码    │     全局自注意力建模
    │   [B,C,H,W]→[B,HW,C]→... │
    └─────────────────────────┘
              |
```

### yolo-improve 筛选结果

| 模块 | 决定 | 原因 |
|------|------|------|
| **AIFI** | ✅ 纳入 | Transformer自注意力，与现有模块互补 |
| ScalSeq | ⚠️ 待评估 | 3D卷积多尺度融合，需Neck重构 |
| CAM | ❌ 跳过 | 与已有 ASPP_Branch 功能重叠 |
| IoU变体 | ❌ 跳过 | 损失函数改动，侵入性强 |
| aLRPLoss | ❌ 跳过 | 自定义autograd，侵入性强 |
| attention_model | ❌ 跳过 | 与已有 Att_ECA + Att_Coord 重叠 |

### Plug-and-play 筛选结果

| 模块 | 决定 | 原因 |
|------|------|------|
| **AFF / iAFF** | ✅ 纳入 | 专门为特征融合设计，替代Concat智能融合双模态 |
| **ULSAM** | ✅ 纳入 | 超轻量子空间注意力，几乎零参数 |
| **StripPool** | ✅ 纳入 | 1D条带池化长程依赖，对细长IR目标友好 |
| Non-local | ❌ 跳过 | 与已有 AIFI 重叠 (O(N²)注意力) |
| CCNet / DANet | ❌ 跳过 | O(N²)或自定义CUDA，太重 |
| scSE / cSE / sSE | ❌ 跳过 | 与已有 SEAttention / CBAM 重叠 |
| SplitAttention | ❌ 跳过 | 重量级Conv替换 |
| GhostModule | ❌ 跳过 | Ultralytics已内置GhostConv |
| DynamicConv / DGC | ❌ 跳过 | 太复杂，侵入训练循环 |
| SPConv | ❌ 跳过 | 源码有bug(未定义变量) |
| BorderAlignment | ❌ 跳过 | 需要自定义CUDA扩展 |
| PSPModule / PyConv / RFB | ❌ 跳过 | 与已有 ASPP / SPPF 重叠 |
