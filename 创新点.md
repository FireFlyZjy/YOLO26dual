## 2026-05-13

### 新增11个注意力模块

在双模态YOLO中融合架构中，于P3/P4/P5三个尺度的Concat融合之后插入注意力模块，用于对融合特征做跨通道/空间重标定。

| 模块 | 论文 | 类型 | 参数量 |
|------|------|------|--------|
| **Att_CBAM** | CBAM (ECCV 2018) | 通道+空间 | +0.05M |
| **Att_Coord** | Coordinate Attention (CVPR 2021) | 通道+坐标编码 | +0.03M |
| **Att_SimAM** | SimAM (ICML 2021) | 无参数3D | +0.00M |
| **Att_CPCA** | CPCA (2023) | 通道先验+多尺度空间 | +0.70M |
| **Att_EMA** | EMA (2023) | 分组交叉空间 | +0.06M |
| **Att_ECA** | ECA-Net (CVPR 2020) | 1D卷积通道 | +0.00M |
| **Att_Shuffle** | ShuffleAttention (2021) | 分组通道+空间 | +0.00M |
| **Att_LSKA** | LSKA (2023) | 大核可分离空间 | +0.41M |
| **Att_Triplet** | Triplet Attention (2020) | 跨维度空间 | +0.00M |
| **Att_GAM** | GAM (2021) | 全局注意力 | +9.83M |
| **Att_ELA** | ELA (2024) | 方向局部注意力 | +0.39M |

**插入策略**: 在双模态Concat融合之后(P3/P4/P5)，SPPF之前。注意力模块学习两种模态特征的互补性权重，抑制融合噪声。

**改进动机**: 双模态特征直接Concat存在信息冗余和模态冲突，注意力机制能自适应地筛选融合后的有效特征，在不同尺度上独立学习可见光-红外的重要性分布。

### 先前已集成的模块（7个）

| 模块 | 来源 | 核心思想 |
|------|------|----------|
| **ACBlock** | ACNet (ICCV 2019) | 非对称卷积，训练时3×3→3×3+3×1+1×3，推理BN融合零开销 |
| **BlazeBlock** | BlazeFace (CVPR 2019) | dw+pw-linear无激活瓶颈，轻量化IR分支 |
| **ASPP_Branch** | DeepLab | 不同膨胀率深度可分离卷积并行，多尺度上下文 |
| **GCBlock** | Context Gating (2017) | 跨通道特征重标定，可学习权重矩阵+BN+sigmoid |
| **C2f_DCN** | DCNv2 (CVPR 2019) | 可变形卷积v2，自适应采样+调制掩码 |
| **SEAttention** | SENet (CVPR 2018) | 全局池化→FC→ReLU→FC→Sigmoid通道重标定 |
| **CSP_ATT** | CSPNet + SE | SPPF替代方案，CSP+多尺度MaxPool+SE |

### 验证脚本

创建 `val_zjy_full.py`，输入模型权重目录即可输出 Precision/Recall/mAP50/mAP50-95/FLOPs/Params/FPS(含Preprocess/Inference/Postprocess/Total Time)，并追加到CSV。

### 验证结果

- 全部11个新注意力模块模型可成功实例化并完成forward pass
- Att_SimAM在FLIR数据集上完成1 epoch训练验证，无错误

## 2026-05-14

### 新增AIFI (RT-DETR Transformer自注意力)

从 `yolo-improve` 目录中筛选，将 RT-DETR 的 AIFI 模块集成到双模态YOLO中，替代 SPPF 层。

| 模块 | 论文 | 类型 | 参数量 |
|------|------|------|--------|
| **Att_AIFI** | RT-DETR (2023) | Transformer自注意力+2D位置编码 | +2.33M |

**改进动机**: SPPF 仅通过固定大小的池化核捕获上下文，无法建模全局长程依赖。AIFI 使用多头自注意力+2D sin-cos位置编码，在 P5 最深特征图上提供全局上下文建模能力，对双模态融合后的特征做跨空间位置的信息交互。

**插入位置**: P5 Concat融合之后，C2PSA之前（替代SPPF），其余结构不变。

**验证结果**: Att_AIFI 模型成功实例化（17.34M params）并通过 forward pass。

### 训练脚本

创建 `train_RGBT_midfuse_testAddModules_withResults.py`，训练完成后自动运行验证并输出 Precision/Recall/mAP50/mAP50-95/FLOPs/Params/FPS 到控制台和 CSV。

### yolo-improve 筛选分析

从 yolo-improve 的 20+ 个改进点中筛选：
- **纳入**: AIFI (全局自注意力)
- **跳过**: CAM(与ASPP重叠), IoU变体(损失函数改动), aLRPLoss(侵入性强), 各种Backbone替换(不在本阶段范围)
- **待评估**: ScalSeq (ASF-YOLO的3D卷积多尺度融合，已实现并验证通过)

### 新增Plug-and-play模块 (4个)

从 `Plug-and-play module` 目录中筛选28个模块，集成4个不重叠且有涨点潜力的模块。

| 模块 | 论文 | 类型 | 参数量 |
|------|------|------|--------|
| **Att_AFF** | AFF (WACV 2021) | 双输入注意力特征融合 | +3.38M |
| **Att_iAFF** | iAFF (WACV 2021) | 迭代注意力特征融合 | +4.95M |
| **Att_ULSAM** | ULSAM (2020) | 超轻量子空间注意力 | +0.01M |
| **Att_StripPool** | SPNet (CVPR 2020) | 1D条带池化长程空间上下文 | +0.28M |

**AFF/iAFF 创新点**: 替代简单的Concat操作，用局部+全局注意力学习可见光与红外特征的逐通道融合权重。这是**唯一专门为特征融合设计的模块**。

**插入策略**:
- AFF/iAFF: 替换P3/P4/P5的Concat，直接融合双模态特征
- ULSAM: Concat之后插入，分组空间softmax注意力
- StripPool: P5 Concat之后替代SPPF，条带池化捕获方向性长程依赖

**验证结果**: 全部4个模块成功实例化并通过forward pass。
